# Карта курса: 30 квестов — «Трубач на коне»

> Оригинальный игровой контент по **структуре** методики Докшицера (комплекс, разминка → вокализ → секвенция).  
> Тексты и упражнения — свои, без копирования нот из книг.  
> Основа: глава 1 «Системы» (OCR ✅) — см. [`extracted/ch1-outline.md`](extracted/ch1-outline.md).

**Режим для новичков (из OCR):** 3–4 сессии по 15–20 мин/день; квесты рассчитаны на 3–7 мин каждый.

**Нарратив:** путешествие всадника-наставника от первого знакомства с трубой до короткой мелодии за месяц.  
**Формат квеста:** 3–7 минут, один навык, XP + разблокировка следующего.
