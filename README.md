# Trumpeter on Horseback

A neo-brutalist web PWA for learning trumpet — short daily quests from your first lip buzz to your first melody. Microphone feedback for sound, pitch, rhythm, and dynamics.

Built with React 19, TypeScript, Vite, Tailwind CSS 4, Zustand, and IndexedDB.

## Features

- 30 quests across 4 weeks
- Russian and English UI
- XP, levels, daily streak, unlockable quest map
- Microphone-based sound, pitch, rhythm, and dynamics exercises
- Installable PWA

## Getting started

```bash
npm install
npm run dev
```

```bash
npm run build
npm run preview
```
